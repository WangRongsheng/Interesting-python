# AI 情感分析

课程以AI技术人员的角度，融入不同实践案例教授机器学习/深度学习/自然语言处理/情感分析等AI技术，
让学生在进行职业体验的同时能掌握基础的AI技术概念，并学会用AI能力解决身边的实际问题。
