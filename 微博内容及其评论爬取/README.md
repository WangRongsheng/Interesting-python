# weibo_spider
# 使用方法
## 1. clone 项目到本地
## 2. 运行文件 python main.py 
## 3. 根据提示，输入信息

### 说明
- 当然支持按照 page 页数爬取某人的微博，page 最大支持 50
- 支持根据微博内容搜索，来爬取某条微博及评论，只支持最近两个月的微博搜索
- 当然，这些都可以配置
- 爬虫速率不要太快，容易被封
